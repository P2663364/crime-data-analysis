# Crime Data Analysis

## Overview
This project analyses crime data using Python, PySpark, Pandas and data visualisation techniques.

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- PySpark
- Pandas
- Data Visualisation
- Exploratory Data Analysis (EDA)

## Tools Used
- Python
- PySpark
- Pandas
- Matplotlib
- Seaborn

## Outcome
The project identifies crime trends, patterns and insights through data processing and visualisation.
